# Robot Controller

Simple grid-based robot navigation simulator in C#.

## Run

```bash
dotnet run
```

## Commands

- **F** - Move forward
- **L** - Turn left
- **R** - Turn right
- **S** - Show status
- **Q** - Quit

## Features

- 10x10 grid navigation
- Random obstacle placement
- Boundary detection
- Collision avoidance
